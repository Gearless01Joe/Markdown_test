# 国自然选题推荐数据清洗

## 项目概述

国自然选题推荐数据清洗模块，实现 `breadth_search` 和 `cited_articles` 字段的数据清洗和标准化处理。

## 快速开始

```python
from data_cleaner import DataCleaner

cleaner = DataCleaner()
cleaner.run()
```

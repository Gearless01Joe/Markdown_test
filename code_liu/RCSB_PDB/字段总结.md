# RCSB PDB 字段总结

基于 `rcsb-restful-api-docs (1).json`，以下内容覆盖 CoreEntry 与 CorePolymerEntity 的全部字段。每个字段均提供中文描述并延续 `缺失字段报告.md` 的缩进格式。其余板块将在下一轮补全。

---

## 一、CoreEntry

audit_author  
描述：条目作者列表，记录每位作者的基本身份信息。

  identifier_ORCID(字符串)：作者的 ORCID 唯一标识。  
  name(字符串)：作者姓名，格式为“姓氏, 名字”并可包含王朝/家族说明。  
  pdbx_ordinal(整数)：作者在作者序列中的排序号。

cell  
描述：晶胞几何参数。

  Z_PDB(整数)：单位晶胞内聚合物链的数量（异源聚合物取最常见链数）。  
  angle_alpha/angle_beta/angle_gamma(数字)：晶胞 α/β/γ 角（度）。  
  formula_units_Z(整数)：晶胞内化学式单位数。  
  length_a/length_b/length_c(数字)：晶胞 a/b/c 轴长度（Å）。  
  pdbx_unique_axis(字符串)：若存在唯一轴（如 P21 的 C 轴），在此标明。  
  volume(数字)：晶胞体积（Å³），按 a·b·c 与角度关系计算。

citation  
描述：条目参考文献列表。

  book_id_ISBN/book_publisher/book_publisher_city/book_title(字符串)：书籍或章节的 ISBN、出版社、地点与书名。  
  coordinate_linkage(字符串)：该引用是否与条目坐标完全对应。  
  country(字符串)：出版国家或地区。  
  id(字符串)：引用记录唯一 ID；`primary` 表示最主要的引用。  
  journal_abbrev/journal_full/journal_id_ASTM/journal_id_CSD/journal_id_ISSN/journal_issue/journal_volume(字符串)：期刊缩写、全名、ASTM/CSD/ISSN 号及期卷信息。  
  language(字符串)：文献语言。  
  page_first/page_last(字符串)：起止页码。  
  pdbx_database_id_DOI(字符串)：DOI。  
  pdbx_database_id_PubMed(整数)：PubMed 号。  
  rcsb_authors(字符串数组)：作者列表（竖线分隔）。  
  rcsb_is_primary(字符串)：是否主引用。  
  rcsb_journal_abbrev(字符串)：规范化期刊缩写。  
  title(字符串)：文献标题。  
  unpublished_flag(字符串)：是否未出版。  
  year(整数)：出版年份。

database_2  
描述：与条目相关的其他数据库编号。

  database_code(字符串)：数据库内的代码。  
  database_id(字符串)：数据库缩写。  
  pdbx_DOI(字符串)：CrossRef 注册的 DOI。  
  pdbx_database_accession(字符串)：扩展访问号。

diffrn  
描述：衍射数据集信息。

  ambient_pressure(数字)：测量时平均压力（kPa）。  
  ambient_temp(数字)：测量时平均温度（K）。  
  ambient_temp_details(字符串)：温控细节。  
  crystal_id(字符串)：指向 `exptl_crystal.id`。  
  crystal_support(字符串)：晶体支撑装置。  
  details(字符串)：衍射测量的特殊说明。  
  id(字符串)：衍射数据集唯一 ID。  
  pdbx_serial_crystal_experiment(字符串)：是否为串行晶体学实验。

diffrn_detector  
描述：衍射探测器信息。

  details(字符串)：探测器特殊说明。  
  detector(字符串)：探测器类别。  
  diffrn_id(字符串)：指向 `diffrn.id`。  
  pdbx_collection_date(字符串)：采集日期。  
  pdbx_frequency(数字)：探测器工作频率（Hz）。  
  type(字符串)：设备型号或名称。

diffrn_radiation  
描述：辐射束信息。

  collimation(字符串)：束流准直或聚焦设置。  
  diffrn_id(字符串)：指向 `diffrn.id`。  
  monochromator(字符串)：单色化方式及晶体信息。  
  pdbx_diffrn_protocol(字符串)：单波长/Laue/MAD。  
  pdbx_monochromatic_or_laue_m_l(字符串)：区分单色或 Laue。  
  pdbx_scattering_type(字符串)：散射类型。  
  pdbx_wavelength/pdbx_wavelength_list(字符串)：波长或波长列表。  
  type(字符串)：辐射性质描述（Siegbahn 记号）。  
  wavelength_id(字符串)：指向 `diffrn_radiation_wavelength.id`。

diffrn_source  
描述：辐射源装置信息。

  details(字符串)：辐射源特殊说明。  
  diffrn_id(字符串)：指向 `diffrn.id`。  
  pdbx_synchrotron_beamline/pdbx_synchrotron_site(字符串)：同步辐射光束线/站点。  
  pdbx_wavelength/pdbx_wavelength_list(字符串)：辐射波长信息。  
  source(字符串)：辐射源类别。  
  type(字符串)：辐射源型号或名称。

em_2d_crystal_entity  
描述：二维晶体几何与处理参数。

  angle_gamma(数字)：γ 角（度）。  
  c_sampling_length(数字)：c 方向倒易格点采样长度。  
  id(字符串)：记录主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。  
  length_a/length_b/length_c(数字)：a/b 轴长度与晶体厚度（Å）。  
  space_group_name_H_M(字符串)：二维平面对称群名称。

em_3d_crystal_entity  
描述：三维晶体单元参数。

  angle_alpha/angle_beta/angle_gamma(数字)：α/β/γ 角。  
  id(字符串)：记录主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。  
  length_a/length_b/length_c(数字)：晶胞轴长。  
  space_group_name(字符串)：空间群名称。  
  space_group_num(整数)：空间群编号。

em_3d_fitting  
描述：3DEM 体密度与原子模型拟合详情。

  details(字符串)：拟合补充说明。  
  id(字符串)：拟合过程 ID。  
  method(字符串)：拟合方法。  
  overall_b_value(数字)：整体 B 因子。  
  ref_protocol(字符串)：精修协议。  
  ref_space(字符串)：拟合在实空间或倒易空间执行。  
  target_criteria(字符串)：拟合质量评价标准。

em_3d_fitting_list  
描述：拟合所用 PDB 链与模型列表。

  3d_fitting_id(字符串)：指向 `em_3d_fitting.id`。  
  details(字符串)：模型详情。  
  id(字符串)：主键。  
  pdb_chain_id(字符串)：参与拟合的链 ID。  
  pdb_chain_residue_range(字符串)：残基范围。  
  pdb_entry_id(字符串)：该模型所属 PDB 代码。

em_3d_reconstruction  
描述：三维重建参数。

  actual_pixel_size/nominal_pixel_size(数字)：实际/标称像素大小（Å）。  
  algorithm/method(字符串)：重建算法与方法。  
  details(字符串)：补充说明。  
  id(字符串)：主键。  
  image_processing_id(字符串)：指向图像处理记录。  
  magnification_calibration(字符串)：放大倍率校准方式。  
  num_class_averages/num_particles(整数)：类别数与粒子数。  
  refinement_type(字符串)：半图处理方式。  
  resolution/resolution_method(数字/字符串)：最终分辨率及判定方法。  
  symmetry_type(字符串)：应用的对称性类型。

em_ctf_correction  
描述：CTF（对比度传递函数）校正信息。

  details(字符串)：CTF 校正附加说明。  
  em_image_processing_id(字符串)：指向 `em_image_processing.id`。  
  id(字符串)：主键。  
  type(字符串)：所用 CTF 校正类型。

em_diffraction  
描述：电子衍射实验记录。

  camera_length(数字)：相机长度（mm）。  
  id(字符串)：主键。  
  imaging_id(字符串)：指向 `em_imaging.id`。  
  tilt_angle_list(字符串)：使用的倾角列表（度，逗号分隔）。

em_diffraction_shell  
描述：电子衍射分辨率壳统计。

  em_diffraction_stats_id(字符串)：指向 `em_diffraction_stats.id`。  
  fourier_space_coverage(数字)：该壳傅里叶空间覆盖率（%）。  
  high_resolution/low_resolution(数字)：该壳高/低分辨率界限（Å）。  
  id(字符串)：主键。  
  multiplicity(数字)：平均测量重复数。  
  num_structure_factors(整数)：结构因子数量。  
  phase_residual(数字)：相位残差（度）。

em_diffraction_stats  
描述：电子衍射整体统计。

  details(字符串)：测量附加说明。  
  fourier_space_coverage(数字)：覆盖率（%）。  
  high_resolution(数字)：最高分辨率。  
  id(字符串)：主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。  
  num_intensities_measured/num_structure_factors(整数)：测得的强度与结构因子数。  
  overall_phase_error/overall_phase_residual(数字)：整体相位误差/残差。  
  phase_error_rejection_criteria(字符串)：相位拒绝准则。  
  r_merge(数字)：Rmerge（%）。  
  r_sym(数字)：Rsym（%）。

em_embedding  
描述：EM 样品包埋信息。

  details(字符串)：包埋步骤描述。  
  id(字符串)：主键。  
  material(字符串)：包埋材料。  
  specimen_id(字符串)：指向 `em_specimen.id`。

em_entity_assembly  
描述：EM 样品组装及子组件。

  details(字符串)：样品或子组件的补充说明。  
  entity_id_list(字符串数组)：涉及的实体 ID 列表。  
  id(字符串)：主键。  
  name(字符串)：样品/子组件名称。  
  oligomeric_details(字符串)：寡聚化信息。  
  parent_id(整数)：父组件 ID（0 表示最上层）。  
  source(字符串)：来源类型。  
  synonym(字符串)：别名。  
  type(字符串)：组件类型。

em_experiment  
描述：EM 实验整体信息。

  aggregation_state(字符串)：样品聚集状态。  
  entity_assembly_id(字符串)：指向 `em_entity_assembly.id`。  
  id(字符串)：主键。  
  reconstruction_method(字符串)：重建方法。

em_helical_entity  
描述：螺旋对称实体参数。

  angular_rotation_per_subunit(数字)：每单元旋转角（度）。  
  axial_rise_per_subunit(数字)：沿轴升高（Å）。  
  axial_symmetry(字符串)：轴对称类型（Cn 或 Dn）。  
  details(字符串)：额外说明。  
  id(字符串)：主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。

em_image_recording  
描述：图像采集记录。

  average_exposure_time(数字)：平均曝光时间。  
  avg_electron_dose_per_image(数字)：每幅图像电子剂量。  
  details(字符串)：记录细节。  
  detector_mode(字符串)：探测器模式。  
  film_or_detector_model(字符串)：探测器型号。  
  id(字符串)：主键。  
  imaging_id(字符串)：对应的显微设置。  
  num_diffraction_images/num_grids_imaged/num_real_images(整数)：衍射图像、网格、真实图像数量。

em_imaging  
描述：显微镜成像参数。

  accelerating_voltage(整数)：加速电压（kV）。  
  alignment_procedure(字符串)：束流对准方法。  
  astigmatism(字符串)：象差信息。  
  c2_aperture_diameter(数字)：C2 聚光镜孔径（µm）。  
  calibrated_defocus_max/min(数字)：经校准的最大/最小离焦（nm）。  
  calibrated_magnification(整数)：校准放大倍率。  
  cryogen(字符串)：成像时使用的冷冻剂。  
  date(字符串)：实验日期。  
  details(字符串)：其他成像细节。  
  detector_distance(数字)：探测器距离（mm）。  
  electron_beam_tilt_params(字符串)：电子束倾斜参数。  
  electron_source(字符串)：电子源类型。  
  id(字符串)：主键。  
  illumination_mode(字符串)：照明模式。  
  microscope_model(字符串)：显微镜型号。  
  mode(字符串)：成像模式。  
  nominal_cs(数字)：物镜球差系数（mm）。  
  nominal_defocus_max/min(数字)：标称最大/最小离焦。  
  nominal_magnification(整数)：标称放大率。  
  recording_temperature_maximum/minimum(数字)：录制时温度范围（K）。  
  residual_tilt(数字)：残余束斜（mrad）。  
  specimen_holder_model/type(字符串)：样品托型号与类型。  
  specimen_id(字符串)：指向 `em_specimen.id`。  
  temperature(数字)：平均载物台温度。  
  tilt_angle_max/min(数字)：样品倾角范围。

em_particle_selection  
描述：粒子筛选/挑选过程。

  details(字符串)：筛选方法与滤波器说明。  
  id(字符串)：主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。  
  num_particles_selected(整数)：筛选出的粒子数。

em_single_particle_entity  
描述：单颗粒分析实体信息。

  id(整数)：主键。  
  image_processing_id(字符串)：指向 `em_image_processing.id`。  
  point_symmetry(字符串)：点群对称符号（Cn、Dn、T、O、I）。

em_software  
描述：EM 流程使用的软件列表。

  category(字符串)：软件用途类别。  
  details(字符串)：软件使用细节。  
  fitting_id/image_processing_id/imaging_id(字符串)：指向对应流程记录。  
  id(字符串)：主键。  
  name(字符串)：软件名称（如 RELION）。  
  version(字符串)：版本号。

em_specimen  
描述：EM 样品对象。

  concentration(数字)：样品浓度（mg/mL）。  
  details(字符串)：制备补充描述。  
  embedding_applied/shadowing_applied/staining_applied/vitrification_applied(字符串)：是否进行包埋/蒸镀/染色/玻璃化。  
  experiment_id(字符串)：指向 `em_experiment.id`。  
  id(字符串)：主键。

em_staining  
描述：染色步骤。

  details(字符串)：染色程序。  
  id(字符串)：主键。  
  material(字符串)：染色材料。  
  specimen_id(字符串)：指向 `em_specimen.id`。  
  type(字符串)：染色类型。

em_vitrification  
描述：玻璃化处理信息。

  chamber_temperature/temp(数字)：玻璃化前舱温 / 浸没温度（K）。  
  cryogen_name(字符串)：使用的冷冻剂。  
  details(字符串)：附加说明。  
  humidity(数字)：相对湿度（%）。  
  id(字符串)：主键。  
  instrument(字符串)：玻璃化设备类型。  
  method(字符串)：玻璃化方法。  
  specimen_id(字符串)：指向 `em_specimen.id`。  
  time_resolved_state(字符串)：事件触发后至玻璃化的时间及说明。

entry  
描述：CIF 数据块入口信息。

  id(字符串)：数据块标识符。  
  ma_collection_id(字符串)：关联模型集合 ID。

exptl  
描述：实验方法概况。

  crystals_number(整数)：用于测量的晶体数量。  
  details(字符串)：强度测量前的特殊实验说明。  
  method(字符串)：实验方法。  
  method_details(字符串)：方法细节。

exptl_crystal  
描述：实验晶体性质。

  colour(字符串)：晶体颜色。  
  density_Matthews(数字)：Matthews 密度（Å³/Da）。  
  density_meas(数字)：实测密度。  
  density_percent_sol(数字)：晶胞溶剂百分比。  
  description(字符串)：晶体形貌及质量描述。  
  id(字符串)：晶体记录 ID。  
  pdbx_mosaicity/pdbx_mosaicity_esd(数字)：各向同性镶嵌角及误差。  
  preparation(字符串)：晶体制备及安装过程。

exptl_crystal_grow  
描述：晶体生长条件。

  crystal_id(字符串)：指向 `exptl_crystal.id`。  
  details(字符串)：特殊生长说明。  
  method(字符串)：生长方法。  
  pH/pdbx_pH_range(数字/字符串)：生长 pH 或 pH 范围。  
  pdbx_details(字符串)：生长过程描述。  
  temp/temp_details(数字/字符串)：生长温度及其控制说明。

ihm_entry_collection_mapping  
描述：IHM（整合模型）集合映射。

  collection_id(字符串)：指向 `ihm_entry_collection.id`。

ihm_external_reference_info  
描述：整合模型外部参考。

  associated_url(字符串)：外部引用对应的下载链接。  
  reference(字符串)：外部引用或 DOI。  
  reference_provider(字符串)：提供方名称。

ma_data  
描述：模型关联数据集。

  content_type(字符串)：数据类型。  
  content_type_other_details(字符串)：其他类型说明。  
  id(整数)：数据集 ID。  
  name(字符串)：数据集名称。

pdbx_SG_project  
描述：结构基因组学项目。

  full_name_of_center(字符串)：中心全称。  
  id(整数)：中心 ID。  
  initial_of_center(字符串)：中心缩写。  
  project_name(字符串)：项目名称。

pdbx_audit_revision_category / pdbx_audit_revision_details / pdbx_audit_revision_group / pdbx_audit_revision_history / pdbx_audit_revision_item  
描述：修订记录（类别、详情、分组、历史、条目）。

  category/group/data_content_type/item/description/details/type(字符串)：各类修订的类别、分组、文件类型、条目、说明。  
  ordinal/revision_ordinal(整数)：记录序号及与历史的关联。  
  provider(字符串)：修订提供方。  
  major_revision/minor_revision/revision_date(整数/字符串)：版本号及发布日期。

pdbx_audit_support  
描述：资助信息。

  country(字符串)：资助国家/地区。  
  funding_organization(字符串)：资助机构。  
  grant_number(字符串)：资助号。  
  ordinal(整数)：序号。

pdbx_database_PDB_obs_spr  
描述：旧版/替换条目信息。

  mechanism_of_action(字符串)：作用机制。  
  synonyms(字符串数组)：同义词/商品名。  
  atc_codes(字符串数组)：ATC 分类编码。  
  drug_categories(字符串数组)：药理类别。  
  drug_products(对象数组)：上市产品（name、labeller、country、approved、started_marketing_on、ended_marketing_on）。  
  affected_organisms(字符串数组)：受影响生物。  
  brand_names(字符串数组)：品牌名。  
  drugbank_id(字符串)：DrugBank ID（若此处重复定义，保持与字段一致）。

drugbank_target  
描述：DrugBank 记录的靶点列表（数组）。

  name(字符串)：靶标名称。  
  organism_common_name(字符串)：靶标来源生物。  
  reference_database_name/reference_database_accession_code(字符串)：参考数据库与 accession（通常为 UniProt）。  
  interaction_type(字符串)：药物对靶点的相互作用类型。  
  target_actions(字符串数组)：药物对靶点的具体作用。  
  seq_one_letter_code(字符串)：靶点氨基酸序列。  
  rcsb_protein_id_list(字符串数组)：关联 RCSB 蛋白（若文档提供）。

至此，CoreDrugbank 字段全部覆盖。

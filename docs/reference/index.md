## 使用建议

1. 查找术语 → 术语表；若缺项请提交 PR。
2. 了解版本背景 → 更新日志；必要时附上截图/链接。
3. 追踪实施进度 → 路线图；确认阶段目标后，在对应 Issue/Project 更新状态。
4. 外部链接统一放在 `resources/external.md`，便于审核与持续更新。

!!! note "Frozen Snapshot"
    发布新版本时，可将 `site/` 产物上传到对象存储并记录链接，确保任意历史版本可被访问与比对。

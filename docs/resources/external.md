## 使用说明

- 仅列出官方或可信链接，若引用博客/论坛需注明来源与更新日期。
- 新增资源需包含「添加原因」与「维护者」信息，并在 PR 中说明。
- 建议每季度检查一次链接有效性，可使用 `mkdocs build --strict` + 外部 Link Checker。*** End Patch
